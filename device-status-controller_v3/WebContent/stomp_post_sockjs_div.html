To read data from a topic:
1. Enter consumer topic id assigned to the stream in the Topic field, stream id in the Stream ID field and click Get Data.
To post message to a topic:
2. Enter producer topic id assigned to the stream in the Topic field, stream id in the Stream ID field, message is added to the Message To Topic and click Get Data.
